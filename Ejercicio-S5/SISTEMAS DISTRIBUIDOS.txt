# Sistemas distribuidos 

PAN -> red personal 
LAN -> red local
WAN -> red mundial
ISP -> se conecta a todas las redes del planeta 
Tarjeta de red -> solo se paga una vez  es el conector que se usa para que puedan conectarse a la red, como la que venden claro, tigo o cualquier distribuidor de internet.
Sistema operativo -> algunas gratis como Linux ootros de pago como windous, iOS.
CDN/cache -> es una copia que se crea despues de la primera busqueda al servidor principal, quitando carga al servicio
